Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir.

  • Parágrafo:
      • Use a tag <p>.

  • Título:
      • Basta usar as tags do <h1> até o <h6>. (variando o tamanho)

  • Imagem:
      • Use a tag <img>.

  • Abreviações:
      • Use a tag <abbr>.

  • Código-fonte:
      • Use a tag <code>.

  • Listas numeradas:
      • Use as tags <ol> e <li>.

  • Lista com demarcadores:
      • Use as tags <ul> e <li>.
