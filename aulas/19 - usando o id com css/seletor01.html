Criando sites com HTML e CSS

Aprendendo HTML

HTML básico

Nesta etapa, você aprenderá os fundamentos do HTML, como estruturar um documento, usar tags essenciais como <html>, <head>, <body>, <p>, <a>, <img> e criar links e listas simples.

HTML intermediário

Aqui, você irá aprofundar seus conhecimentos, aprendendo a utilizar formulários, tabelas, iframes, e atributos como class e id para deixar a estrutura do site mais organizada e semântica.

HTML avançado

Nesta fase, serão abordados conceitos mais complexos, como acessibilidade, SEO, uso de metadados, elementos semânticos modernos (<article>, <section>, <aside>), além de integração com APIs e elementos multimídia.

Aprendendo CSS

CSS básico

Você vai começar aprendendo como aplicar estilos às suas páginas HTML usando seletores, cores, fontes, tamanhos, margens, paddings e bordas. Também entenderá a diferença entre estilos inline, internos e externos.

CSS intermediário

Nesse nível, você vai explorar layouts com Flexbox, posicionamento de elementos, pseudo-classes, pseudo-elementos e a utilização de variáveis CSS. Isso vai te permitir criar páginas mais organizadas e responsivas.

CSS avançado

Por fim, você estudará técnicas avançadas como Grid Layout, animações, transições, media queries para responsividade, e boas práticas de arquitetura CSS como BEM (Block Element Modifier) e organização modular.
