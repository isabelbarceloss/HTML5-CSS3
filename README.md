# Curso de HTML5 e CSS3 - Curso em Vídeo (Módulo 1 e 2).
Esse repositório reúne todos os exercícios, testes e projetos que fiz durante as aulas.

## `Módulo 1:`
- [x] Entendimento da estrutura básica de uma página web;
- [x] Semântica HTML5 (usar tags com significado correto);
- [x] Trabalhar com textos, títulos, listas, links e âncoras;
- [x] Inserir multimídia básica (imagens, vídeos, áudios);
- [x] Introdução ao CSS: aplicar estilos simples, ajustar cores, fontes e dar forma à estrutura com estilos básicos.

## `Módulo 2:`
- [x] Fundamentos de design visual para web: conceitos como tipografia, escolha de cores e estética geral da página;
- [x] Uso de seletores CSS, classes e IDs para estilização mais modular e organizada;
- [x] Entendimento do modelo de caixas (box model), margin, padding, border;
- [x] Estruturação de layouts mais organizados, com atenção à estética e responsividade básica (preparar pra diferentes tamanhos de tela, mesmo que simples).
